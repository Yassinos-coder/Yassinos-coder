## 👨‍💻 About Me

I started coding at 14 out of pure curiosity, and it turned into a career building production systems. I work across the full stack — from React and Next.js frontends to microservice APIs on Node.js and Supabase, deployed on Railway. I've shipped integrations with platforms like TutorCruncher, Pabbly, and Slack, and I care about clean architecture, type safety, and APIs that are actually pleasant to use.

Outside of work, I'm into cars, bikes, gaming, and streaming. If that sounds like your kind of developer, let's build something.

- 🌍  Based in **Casablanca, Morocco**
- 🖥️  Portfolio at [yassinoscoder.codes](http://yassinoscoder.codes/)
- ✉️  Reach me at [castroyassine0@gmail.com](mailto:castroyassine0@gmail.com)
- 🧠  Currently deepening expertise in **distributed systems** and **AI integrations**
- 🤝  Open to collaborating on web, mobile, or API projects
